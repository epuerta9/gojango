Getting Started

This example demonstrates how to integrate databases into your Gojango application. The database layer supports:

// Database configuration config := db.PostgresConfig("localhost", "mydb", "user", "password") // or config := db.SQLiteConfig("./app.db") // Connection management conn, err := db.Open(config) migrator := db.NewMigrator(conn, "migrations")

Migration Management

Use the CLI to manage your database schema:

# Create a new migration gojango db makemigration create_users # Apply pending migrations gojango db migrate # Check migration status gojango db showmigrations # Rollback last migration gojango db rollback
